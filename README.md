# Reproductive Health Chatbot

### Purpose
A bot that helps address gaps in reproductive health education, especially in restricted areas
Provides a safe, confidential resource for teenagers and YA

### Features
This health chatbot offers tailored, reliable advice on topics like birth control, STIs, abortion, and consent
Chatbox compliments traditional education systems and supports informed decision-making 

### Tech Stack:
NPM, Flask, React, Langchain, Neo4j

## Project Setup and Running

### Prerequisites

- Node.js v18x and npm (If not installed, download from [Node.js official website](https://nodejs.org/))

### Steps

1. **Clone the Repository**:
   
   ```bash
   git clone https://github.com/ilsilfverskiold/react-langchain-chatbot.git
   cd react-langchain-chatbot

3. **Install Dependencies**:
   
   ```bash
   npm install

4. **Set Up OpenAI API Key**:

- Obtain your OpenAI API key.
- Create a .env file in the root directory and add your OpenAI key.

   ```bash
   REACT_APP_OPEN_AI_API_KEY=your_openai_api_key

4. **Start the Frontend Server**:
   
   ```bash
   npm start

5. **Start the Backend Server**:
   
   ```bash
   python3 app.py
